About us

Code Academy is an education innovation hub dedicated to the vision that every student in every school has the opportunity to learn computer science. We expand access to computer science in schools. Code Academy is the leading provider science curriculum in South Africa. Code Academy also organizes the annual CodeFest campaign, which has engaged more than 15% of all students in South Africa. Code Academy is supported by generous donors including Microsoft, Amazon , Google and many others.
